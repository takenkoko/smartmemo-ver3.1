Django・Bootstrap・CSSで構築したシンプルなメモ管理Webアプリです。

## Features(主な機能)
- Create memo
- Edit memo
- Delete memo
- Search memo
- Category support
- Category badges
- User authentication (Login / Logout)
- User-specific memo management
- Bootstrap UI
- Responsive navigation bar
- Custom CSS
- Login page UI improvements

## Tech Stack
- Python
- Django
- Bootstrap 5
- CSS3
- SQLite 
- Git
- GitHub

# SmartMemo Ver1.0

DjangoとBootstrapで構築したシンプルなメモアプリです。

## Features
- Create memo
- Edit memo
- Delete memo
- Bootstrap UI

## Tech Stack
- Python
- Django
- Bootstrap 5
- SQLite 
- Git
- GitHub

## Future Plans
- Categories
- User authentication
- Responsive UI improvements

  ## Ver2.0で追加する機能

-  検索機能

- ログイン機能（SQLite ➡　PostgreSQL）

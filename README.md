# SmartMemo Ver3.0

【アプリのスクリーンショット】
![SmartMemo](screenshots/home_sample2.0.png)

DjangoとBootstrapで構築したシンプルなメモアプリです。

## Features(主な機能)
- Create memo
- Edit memo
- Delete memo
- Search memo
- Category support
- Category badges
- User authentication (Login / Logout)
- User-specific memo management
- Bootstrap UI
- Responsive navigation bar

## Tech Stack
- Python
- Django
- Bootstrap 5
- SQLite 
- Git
- GitHub

## Future Plans
